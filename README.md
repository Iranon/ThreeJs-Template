# ThreeJs-Template

It implements three.module.js inside the html file and it imports Loaders(glTF and OBJ), Controls and custom GLSL shaders (as .js files inside the shaders folder).


ThreeJsTemplate
├───build
├───css
├───jsm
│   ├───controls
│   │   └───experimental
│   └───loaders
│       ├───lwo
│       └───obj2
│           ├───bridge
│           ├───shared
│           ├───utils
│           └───worker
│               ├───main
│               └───parallel
├───models
├───shaders
└───textures
